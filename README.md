# Emotion Detection using MediaPipe and FACS

This project implements a rule-based emotion detection system using MediaPipe Face Landmarker and Paul Ekman's Facial Action Coding System (FACS).

## Features
- Real-time webcam-based emotion detection
- Detects:
  - Happy
  - Angry
  - Surprised
  - Neutral
- Uses facial landmarks from MediaPipe
- Rule-based classification using facial measurements

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
